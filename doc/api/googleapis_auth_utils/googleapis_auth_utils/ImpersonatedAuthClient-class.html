ImpersonatedAuthClient

ImpersonatedAuthClient class

Impersonated service account credentials.

This class allows credentials issued to a user or service account to impersonate another service account. The source project using impersonated credentials must enable the "IAMCredentials" API. Also, the target service account must grant the originating principal the "Service Account Token Creator" IAM role.

This is the Dart equivalent of the Node.js Impersonated class.
